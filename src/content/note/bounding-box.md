- A bounding box is a kind of box that is flexible and it kind of wrap around any object. Means if my object is a star then bounding box will create a rectangle around my star object by touching all maximum edges of star. Let's say if out of 5 edge, 3 edges are bigger and 2 are smaller then bounding box will draw in such a way that it will touch 3 edges.
- Bounding box is used to identify a shape of object like how big it is or small. Mostly in itext pdf work and in AI object determine.
